“Quotations From Chairman Mao Zedong” Unix Fortune File

Download: Little-Red-Book

For the Marxist-Leninist-Maoist unix/fortune users (all 3 of you!), I present the entire Quotations from Chairman Mao Zedong book in a fortune(6) ready format.

Download the file above, then run the following command in the download directory to install:

strfile Little-Red-Book Little-Red-Book.dat; sudo mv Little-Red-Book /usr/share/games/fortunes/; sudo mv Little-Red-Book.dat /usr/share/games/fortunes/

Note: /usr/share/games/fortunes/ may not be the directory used by the version of fortune your distribution ships. Check the "FILES" section in your man page if this doesn't work.
